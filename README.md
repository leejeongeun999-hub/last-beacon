# Last Beacon: Orbit Defense

Portrait iPhone tower defense game built with SwiftUI and SpriteKit.

## Local build

```sh
xcodegen generate
xcodebuild test -project LastBeacon.xcodeproj -scheme LastBeacon -destination 'platform=iOS Simulator,id=1FDC7842-B127-4CB3-8A00-F56311CFA433'
```

Product identity and release gates are defined in `docs/superpowers/specs/2026-07-19-last-beacon-design.md`.
